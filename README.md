# ささっと交換ツイ
X（旧：Twitter）でのグッズ交換募集や成立後のDMを簡単に作成できるツールです。
グッズ交換初心者の方でも項目を入力するだけで、交換募集のポストを作成することができます。

## 使い方
1. 各項目を入力することで、自動でテンプレートに反映されます。
2. 「コピーしてXを開く」を押すと、文章がコピーされた状態でXが起動します。

〇 梱包方法、マイ定型文はテンプレート保存ができます。次回以降スムーズな取引ができるように活用してください。

※手渡し場所、住所はブラウザ上にデータが保存され、2度目以降は自動で入力されます。
　あくまであなたのブラウザにデータが保持されるため、運営側が情報を確認することはできません。ご安心ください。
